## Important Notes

### Query Guidelines
- Be specific about what you want to analyze
- Mention specific cryptocurrencies if you have preferences
- Ask about timeframes if you're interested in particular periods
- Focus on learning and understanding rather than predictions

### What to Expect
- Educational insights about market dynamics
- Analysis of multiple factors affecting prices
- Neutral, objective perspectives
- Risk awareness and disclaimers
- No financial advice or trading signals

### Limitations
- Analysis is educational and conceptual
- No real-time trading recommendations
- No price predictions or forecasts
- Focus on understanding, not timing
- Emphasis on risk education

## Example Usage

### Basic Query
```
POST /analyze
{
  "query": "What are the current market trends for Bitcoin?"
}
```

### Specific Assets
```
POST /analyze
{
  "query": "Compare market conditions for Bitcoin and Ethereum",
  "assets": ["bitcoin", "ethereum"]
}
```

### Complex Analysis
```
POST /analyze
{
  "query": "How do macroeconomic factors and on-chain metrics influence Bitcoin market conditions?",
  "assets": ["bitcoin"]
}
```

Remember: MacroChain AI is designed for educational purposes only. Use these queries to learn about market dynamics, not to make investment decisions.
